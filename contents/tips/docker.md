# Docker Command

<!-- tabs:start -->

### **Show**

```bash
$ docker ps
```

```bash
$ docker images
```

### **Load**

```bash
$ docker load -i [image_file_name]
```

### **BackUp**

```bash
$ docker commit [container_name] [new_image_name]
```

```bash
$ docker  save -o [new_file_name].tar [image_name]
```
