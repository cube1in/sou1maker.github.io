# Docker Command

<!-- tabs:start -->

### **Build**

```bash
$ docker build -t [name]:[tag] .
```

### **BackUp**

```bash
$ docker commit [container_id] [new_image_name]
```

```bash
$ docker  save -o [new_file_name].tar [image_id]
```
